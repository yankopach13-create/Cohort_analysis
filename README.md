# Когортный анализ (Cohort Analysis)

Приложение для автоматизации когортного анализа с интерфейсом на Streamlit.

Проект для анализа когорт клиентов, их возвращаемости и оттока.

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Запустите `run_app.bat` - скрипт автоматически создаст виртуальное окружение и установит все зависимости
3. Приложение откроется в браузере автоматически

## Использование

1. Запустите приложение через `run_app.bat`
2. Загрузите Excel файл (.xlsx или .xls) через интерфейс
3. Данные будут загружены и готовы для дальнейшего анализа

## Зависимости

- streamlit - веб-интерфейс
- pandas - работа с данными
- openpyxl - чтение Excel файлов
